# MachineLearningEngineerHiringTest
Sample code for test

The objective of this project is to design and demonstrate a production-grade Retrieval-Augmented Generation (RAG) system and a supporting Scalable Data Platform within the Azure Ecosystem.

Key Goals:

Semantic Intelligence: Moving beyond keyword search to understand the intent and meaning of employee queries.
Fact-Grounded Responses: Eliminating hallucinations by forcing the AI to provide verifiable citations for every claim.
Zero-Trust Security: Ensuring that the AI strictly respects existing organizational permissions (Identity-based security trimming).
Enterprise Governance: Implementing rigorous evaluation, PII protection, and audit logging to meet compliance standards.
